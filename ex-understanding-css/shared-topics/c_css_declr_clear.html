The 'clear' property: How to position boxes next to floats

The CSS 'clear' property helps web designers conrowol block-level elients in relationship to 'floated' boxes.

According to the W3 specification on CSS2, the 'clear' property "indicates which sides of an elient's box(es) may not be adjacent to an earlier floating box." In other words, the 'clear' property only applies to block-level elients -- not floated elients within itself or in other block formatting contexts.
